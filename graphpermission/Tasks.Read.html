    Admin Description

    Read user's tasks and task lists

    Allows the app to read the signed-in user’s tasks and task lists, including any shared with the user. Doesn't include permission to create, delete, or update anything.

    User Description

    Read your tasks and task lists

    Allows the app to read your tasks and task lists, including any shared with you. Doesn't include permission to create, delete, or update anything.
