SecurityIncident.ReadWrite.All

Allows the app to read and write security incidents, on behalf of the signed-in user.

Graph Methods

Type: A = Application Permission, D = Delegate Permission

Ver Type Method
V1 A,D GET /security/incidents
V1 A,D GET /security/incidents/{incidentId}
V1 A,D PATCH /security/incidents/{incidentId}
V1 A,D POST /security/incidents/{incidentId}/comments

Delegate Permission
Id 128ca929-1a19-45e6-a3b8-435ec44a36ba
Consent Type Admin
Display String Read and write to incidents
Description Allows the app to read and write security incidents, on behalf of the signed-in user.

Application Permission
Id 34bf0e97-1971-4929-b999-9e2442d941d7
Display String Read and write to all security incidents
Description Allows the app to read and write to all security incidents, without a signed-in user.
