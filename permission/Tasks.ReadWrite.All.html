Tasks.ReadWrite.All

Allows the app to create, read, update and delete all users’ tasks and task lists in your organization, without a signed-in user

For an app to access data in Microsoft Graph, the user or administrator must grant it the permissions it needs. This article lists all the Microsoft Graph APIs and your tenant data that can be accessed by the application (vendor/developer) if you consent to the Tasks.ReadWrite.All permission.
