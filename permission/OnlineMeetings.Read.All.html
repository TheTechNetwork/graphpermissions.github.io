OnlineMeetings.Read.All

For an app to access data in Microsoft Graph, the user or administrator must grant it the permissions it needs. This article lists all the Microsoft Graph APIs and your tenant data that can be accessed by the application (vendor/developer) if you consent to the OnlineMeetings.Read.All permission.

Allows the app to read online meeting details in your organization, without a signed-in user.

Category Application Delegated
Identifier c1684f21-1984-47fa-9d61-2dc8c296bb70 -

Resources

Granting this permission allows the calling application to access the following information from your tenant.

  • callRecording
  • callTranscript
  • meetingregistrant
  • meetingregistration
  • onlineMeeting
  • virtualEventWebinar
