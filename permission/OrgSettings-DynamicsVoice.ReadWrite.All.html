OrgSettings-DynamicsVoice.ReadWrite.All

For an app to access data in Microsoft Graph, the user or administrator must grant it the permissions it needs. This article lists all the Microsoft Graph APIs and your tenant data that can be accessed by the application (vendor/developer) if you consent to the OrgSettings-DynamicsVoice.ReadWrite.All permission.

Allows the app to read and write organization-wide Dynamics customer voice settings on behalf of the signed-in user.
