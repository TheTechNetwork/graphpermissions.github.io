# LearningAssignedCourse.ReadWrite.All

> Allows the app to create, update, read and delete all assignments in the organization's directory, without a signed-in user.
## Graph Methods

Type: A = Application Permission, D = Delegate Permission

|Ver|Type|Method|
|-------|----|------|
|V1|A|[DELETE /employeeExperience/learningProviders/{registrationId}/learningCourseActivities/{id}](https://docs.microsoft.com/graph/api/learningcourseactivity-delete?view=graph-rest-1.0&tabs=http)|
|V1|A|[PATCH /employeeExperience/learningProviders/{registrationId}/learningCourseActivities/{learningCourseActivityId}](https://docs.microsoft.com/graph/api/learningcourseactivity-update?view=graph-rest-1.0&tabs=http)|
|V1|A|[POST /employeeExperience/learningProviders/{registrationId}/learningCourseActivities](https://docs.microsoft.com/graph/api/employeeexperienceuser-post-learningcourseactivities?view=graph-rest-1.0&tabs=http)|
