# EduRoster.ReadWrite

> Allows the app to read and write the structure of schools and classes in an organization's roster and education-specific information about users to be read and written on behalf of the user.
## Graph Methods

Type: A = Application Permission, D = Delegate Permission

|Ver|Type|Method|
|-------|----|------|
|V1|D|[GET /education/classes/delta](https://docs.microsoft.com/graph/api/educationclass-delta?view=graph-rest-1.0&tabs=http)|
|V1|D|[GET /education/schools/delta](https://docs.microsoft.com/graph/api/educationschool-delta?view=graph-rest-1.0&tabs=http)|
|V1|D|[GET /education/users/delta](https://docs.microsoft.com/graph/api/educationuser-delta?view=graph-rest-1.0&tabs=http)|
