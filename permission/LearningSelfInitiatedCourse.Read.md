# LearningSelfInitiatedCourse.Read

> Allows the app to read data for the learner's self-initiated courses in the organization's directory, on behalf of the signed-in user.
## Graph Methods

Type: A = Application Permission, D = Delegate Permission

|Ver|Type|Method|
|-------|----|------|
|V1|D|[GET /employeeExperience/learningCourseActivities/{Id}](https://docs.microsoft.com/graph/api/learningcourseactivity-get?view=graph-rest-1.0&tabs=http)|
