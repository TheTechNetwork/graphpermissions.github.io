AttackSimulation.ReadWrite.All

Allows the app to read, create, and update attack simulation and training data for an organization for the signed-in user.

Graph Methods

Type: A = Application Permission, D = Delegate Permission

Ver Type Method
V1 A,D GET /security/attackSimulation/endUserNotifications
V1 A,D GET /security/attackSimulation/landingPages/{landingPageId}
V1 A,D GET /security/attackSimulation/loginPages/{loginPageId}
V1 A,D GET /security/attackSimulation/simulations/{simulationId}/landingPage
V1 A,D GET /security/attackSimulation/simulations/{simulationId}/loginPage
V1 A,D GET /security/attackSimulation/trainings/{trainingId}
V1 A,D GET /security/attackSimulation/trainings/{trainingId}/languageDetails/{trainingLanguageDetailId}?$filter=locale eq 'locale'
