Member.Read.Hidden

Allows the app to read the memberships of hidden groups and administrative units on behalf of the signed-in user, for those hidden groups and administrative units that the signed-in user has access to.

Graph Methods

Type: A = Application Permission, D = Delegate Permission

Ver Type Method
V1 A GET /education/classes/{id}/members
V1 A POST /education/classes/{id}/members/$ref

Delegate Permission
Id f6a3db3e-f7e8-4ed2-a414-557c8c9830be
Consent Type Admin
Display String Read hidden memberships
Description Allows the app to read the memberships of hidden groups and administrative units on behalf of the signed-in user, for those hidden groups and administrative units that the signed-in user has access to.

Application Permission
Id 658aa5d8-239f-45c4-aa12-864f4fc7e490
Display String Read all hidden memberships
Description Allows the app to read the memberships of hidden groups and administrative units without a signed-in user.
