PrintJob.ReadBasic.All

Allows the application to read the metadata of print jobs on behalf of the signed-in user. Does not allow access to print job document content.

Graph Methods

Type: A = Application Permission, D = Delegate Permission

Ver Type Method
Beta A,D GET /print/printers/{id}/jobs
V1 A,D GET /print/printers/{id}/jobs/{id}
V1 A,D GET /print/printers/{printerId}/jobs
Beta A,D GET /print/shares/{id}/jobs
V1 A,D GET /print/shares/{printerShareId}/jobs

Delegate Permission

Consent Type Admin

Description Allows the application to read the metadata of print jobs on behalf of the signed-in user. Does not allow access to print job document content.
