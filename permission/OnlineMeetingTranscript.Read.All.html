OnlineMeetingTranscript.Read.All

Allows the app to read all transcripts of online meetings, on behalf of the signed-in user.

Graph Methods

Type: A = Application Permission, D = Delegate Permission

Ver Type Method
V1 A,D GET /me/onlineMeetings/{meetingId}/transcripts/{transcriptId}
V1 A,D GET /me/onlineMeetings/{online-meeting-id}/transcripts
V1 A,D GET /users/{user-id}/onlineMeetings/{online-meeting-id}/transcripts
V1 A,D GET /users/{userId}/onlineMeetings/{meetingId}/transcripts/{transcriptId}

Delegate Permission

Consent Type Admin

Description Allows the app to read all transcripts of online meetings, on behalf of the signed-in user.
