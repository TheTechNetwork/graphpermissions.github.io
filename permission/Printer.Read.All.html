Printer.Read.All

Allows the application to read printers on behalf of the signed-in user.

For an app to access data in Microsoft Graph, the user or administrator must grant it the permissions it needs. This article lists all the Microsoft Graph APIs and your tenant data that can be accessed by the application (vendor/developer) if you consent to the Printer.Read.All permission.

Resources

Granting this permission allows the calling application to access the following information from your tenant.

  • printer
  • printerCapabilities
  • printService
  • printServiceEndpoint
  • printTaskTrigger
