About

While the Microsoft Graph reference page provides a summary of all the permissions, it's not easy to find out the APIs and data that are made available when a consent is provided to an application.

This site will help you quickly get a list of all the APIs for a given Graph permission scope.

How it works

This site is generated by parsing the Graph API reference docs and extracting the permissions and the APIs that are available for each permission.

An automation runs once a day to update the site with the latest APIs and permissions.
